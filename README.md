# Service System Project

### Purpose

This project is meant to explore the following areas:

- Rust Language
- Rust Backend
- Axum Framework
- Middleware
- CRUD operations
- REST API
- MySQL database
- Database Drivers
- Authentication
- Passwordless JWT
- Password JWT
- Role Based System

### Features
This project is a service system where you can do the following:

- Manage machines
- Manage users
- Manage tasks
- Manage workers

### Techstack

#### Backend

- Rust (language)
- Axum (framework)
