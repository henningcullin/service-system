# Service System Project

### Purpose

This project is meant to explore the following areas:

- Rust Language
- Rust Backend
- Axum Framework
- Middleware
- CRUD operations
- REST API
- Server Sent Events
- PostgreSQL database
- Database Drivers
- Authentication
- Passwordless JWT
- Password JWT
- Role Based System
- Shadcn UI

### Features
This project is a service system where you can do the following:

- Manage machines
- Manage users
- Manage tasks
- Manage reports
- Manage facilities

### Techstack

#### Backend

- Rust (language)
- Axum (framework)
- PostgreSQL (database)

#### Frontend

- Svelte
- Shadcn UI
